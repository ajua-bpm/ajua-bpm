🛡️ Protección de Datos — Sistema Anti-Pérdida
📸 Nivel 1 — Snapshots Locales (cada hora, últimas 24h)
Guardados automáticamente en este dispositivo. Funcionan sin internet.
☁️ Nivel 2 — Backup Firebase (documento separado, cada hora)
Copia independiente en ajua_bpm/backup_auto — separada de los datos en uso. Nunca se sobreescribe si tiene más datos.
💾 Nivel 3 — Respaldo Manual JSON
Descarga un archivo JSON completo a tu computadora. Guárdalo en Drive, WhatsApp, correo — donde quieras.
🔒 Guard activo: El sistema bloquea automáticamente cualquier guardado que tenga menos del 30% de registros del último snapshot — previniendo pérdidas accidentales.
